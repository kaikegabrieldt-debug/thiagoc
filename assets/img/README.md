# Imagens

Coloque aqui as fotos reais e o site usa automaticamente (os nomes abaixo já estão referenciados no `index.html`).
Enquanto não houver os arquivos, o site mostra placeholders elegantes — nada quebra.

| Arquivo            | Onde aparece            | Dimensão sugerida | Observações                                         |
| ------------------ | ----------------------- | ----------------- | --------------------------------------------------- |
| `dr-hero.jpg`      | Hero (topo)             | 900 × 1100 px     | Retrato vertical, fundo neutro, boa iluminação      |
| `dr-sobre.jpg`     | Seção "Sobre"           | 800 × 1000 px     | Retrato ou foto em consultório/clínica              |
| `og-image.jpg`     | Prévia ao compartilhar  | 1200 × 630 px     | Foto + nome; usada no WhatsApp/redes (Open Graph)   |

## Como ativar uma foto
1. Salve o arquivo nesta pasta com o nome exato da tabela.
2. No `index.html`, troque o bloco de placeholder pela tag `<img>` indicada nos comentários `<!-- FOTO: ... -->`.

## Dicas
- Prefira JPG (fotos) com peso < 300 KB cada — comprima em https://squoosh.app.
- Evite imagens com texto embutido (ruim para acessibilidade e responsividade).
- Direitos de uso: utilize apenas fotos próprias ou licenciadas.
